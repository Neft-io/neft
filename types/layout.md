Positioning
===========

This category contains items used to position elements.

```style
Column {
\  spacing: 5
\
\  Rectangle { color: 'blue'; width: 50; height: 50; }
\
\  Row {
\    spacing: 15
\    Rectangle { color: 'green'; width: 20; height: 50; }
\    Rectangle { color: 'red'; width: 50; height: 20; }
\  }
}
```
