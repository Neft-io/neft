Response Error

API ▸ Networking ▸ Response ▸ Response Error

Source

Error(status, message)

Parameters:
  • status — Integer — optional
  • message — String — optional
Returns:
Error

It works as a standard Javascript Error class, but provides an extra status value.

Access it with:

var Networking = require('networking');
var ResponseError = Networking.Response.Error;

Source

Error.RequestResolve(request)

Parameters:
  • request — Networking.Request
Returns:
RequestResolve

This error is sent if the request can't be resolved, because no proper handler which can handle the request can be found.

Access it with:

var Networking = require('networking');
var RequestResolveResponseError = Networking.Response.Error.RequestResolve;
