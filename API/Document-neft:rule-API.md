# neft:rule

Tag used in the parsing process.
Performs some actions on found elements in the parent element.

## Table of contents
* [neft:rule](#neftrule)

## attrs

Adds attributes if not exists.

```xml
<neft:rule query="input[type=string]">
  <attrs class="specialInput" />
</neft:rule>
```
