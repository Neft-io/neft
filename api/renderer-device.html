Device.platform

Type:
Boolean
Default value:
'Unix'

Possible values are:

  • Android,
  • iOS,
  • BlackBerry,
  • WindowsCE,
  • WindowsRT,
  • WindowsPhone,
  • Linux,
  • Windows,
  • Unix,
  • OSX.
Text {
    text: 'You are using: ' + Device.platform
    font.pixelSize: 30
}
